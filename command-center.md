# Command Center

The Command Center is the dashboard we build for clients. It surfaces every automated system in one place: AI agent activity, lead flow, appointments booked, revenue recovered, response rates, review generation, and sync status across connected tools.

## What the dashboard shows

- **Leads Today.** Real-time count of new inquiries captured by the response agents, with day-over-day comparison.
- **Appointments Booked.** Number of meetings and service calls on the calendar, including auto-booked conversions from SMS and email agents.
- **Revenue Recovered.** Dollar value generated by automations that would otherwise have been missed: lead follow-ups, reactivations, abandoned inquiries.
- **Response Rate.** Percentage of inquiries answered by AI agents within target response time, with average seconds-to-first-response.
- **Leads This Week.** Seven-day bar chart of lead volume by day, color-coded by source.
- **Recent Activity.** Live feed of every AI agent action: chat captures, schedule bookings, review requests, follow-up emails, back-office syncs, CRM updates.
- **Agent Performance.** Month-to-date tallies for each agent: Chat, Schedule, Follow-Up, Back Office, Review.
- **Connected Systems.** Status indicators showing which tools are online and synced (CRM, calendar, billing, messaging).

## What it is built on

Custom dashboards like this are built per client on top of Supabase, n8n, and the business's existing systems. Stats update live as agents process inquiries. No data is stored beyond what the automations need to operate. Access is OAuth-authenticated and scoped to the client.
